# Dataset: Wikipedia Page Traffic

## Описание
Ежедневная статистика просмотров страницы Wikipedia за 2018-2019 годы.

## Источник
Kaggle: Wikipedia Web Traffic 2018-19

## Выбранная страница
- **Оригинальное название**: バック・トゥ・ザ・フューチャー_ja.wikipedia_desktop_all-agents
- **Английское название**: Back to the Future (Japanese Wikipedia)

## Структура данных
Файл: `data/processed/time_series.csv`

| Столбец | Тип | Описание |
|---------|-----|----------|
| date | datetime | Дата измерения |
| traffic | float64 | Количество просмотров за день |
| page_name | string | Название страницы |

## Характеристики
- Период: 2018-01-01 - 2019-12-31 (730 дней)
- Пропуски: 0
- Диапазон: 232 - 3,726 просмотров/день
- Среднее: 458.72 просмотров/день
- Сезонность: Явная недельная

## Подготовка
Данные готовы для алгоритма экспоненциального сглаживания Holt-Winters.
